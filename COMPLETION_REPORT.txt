🚀 READY TO RUN COMMANDS

Run Frontend:

  API will be available at: http://localhost:8000

Run Tests:
  $ flutter test
  $ flutter analyze

Build for Production:
  $ flutter build apk --release     # Android
  $ flutter build ios --release     # iOS
  $ flutter build web --release     # Web
